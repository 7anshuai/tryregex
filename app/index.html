Getting started with regex

This interactive tutorial serves as an introduction to regular expressions, specifically regular expressions in JavaScript (which are quite different to regular expressions in other languages).

The console to the left is just a JavaScript console. Type setName('Your name') (substituting your name in) to get started with the tutorial.
